Learn Chinese effortlessly while browsing! LingLook is a browser extension that instantly looks up Chinese words as you hover over them on any webpage!

<strong>Features</strong>
<ul>
    <li>Instant Chinese word lookup in any web page by hovering over text</li>
    <li>Support for both simplified and traditional Chinese characters</li>
    <li>Word definitions with pronunciation (pinyin, zhuyin/bopomofo), HSK level, and TOCFL level</li>
    <li>Detailed character information: stroke order animation, frequency, radical, origin, etc.</li>
    <li>Quick word copy and Google Images search</li>
    <li>Support for keyboard shortcuts</li>
    <li>Customizable appearance and behavior</li>
    <li>Supports English, Vietnamese and partial support for French dictionaries</li>
</ul>

<strong>How to use</strong>
<ul>
    <li>For desktop users:
        <ul>
            <li>Pin LingLook icon to your toolbar for easy access. You can click on it to enable/disable the popup function, or press the default shortcut Alt+O (or Option+O on Mac). You can configure this shortcut in the extension's settings page.</li>
            <li>Use the mouse to hover over a Chinese word, and a popup window will appear.</li>
            <li>When you first hover over a word, the popup window will appear with a dotted border. After 400ms, the popup will change to have a solid outline. From here, you can move the mouse into the popup to change tabs or select entries.</li>
            <li>You can also configure the popup to show only when you hold a key like Control. In that case, the popup will not be interactive until you release that key. After you release the key, it will immediately become interactive.</li>
        </ul>
    </li>
    <li>For mobile users:
        <ul>
            <li>You can enable/disable the extension by tapping on the extension icon.</li>
            <li>After enabling, a small puck will appear on your webpage. Move the puck around and look up Chinese words!</li>
            <li>Puck controls:
                <ul>
                    <li>Single-tap on the puck: Toggle lookup function.</li>
                    <li>Double-tap on the puck: Switch lookup direction (up/down).</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>Inside the popup window, you can see the word's definition, pronunciation (pinyin or zhuyin), HSK level, TOCFL level.</li>
    <li>Click on the word, and you can copy it to clipboard or search it with Google Images.</li>
    <li>Switch to the "Character" tab, here you can see detailed character information like stroke order animation, frequency, radical, origin, etc.</li>
    <li>Customize the appearance and behavior of LingLook by clicking on the settings gear icon in the popup window.</li>
</ul>

<strong>Troubleshooting</strong>
If for some reason the extension does not work as expected, try enabling/disabling the extension, reloading the page, or restarting the browser. If you want to report an issue or have any other suggestions, please open an issue on LingLook's GitHub repository <a href="https://github.com/ph0ngp/linglook/issues" title="LingLook GitHub Issues">here</a>.
